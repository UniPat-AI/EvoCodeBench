Difficulty analysis

This is a long (13-round) greenfield data-engineering build: eventq, a CLI that turns raw event logs into clean, currency-normalized summaries — parsing, deduplication, sessionization, currency conversion via FX rates, sharded reporting, and reproducible run provenance. The 13-round chain accumulates a deep contract surface, and each round's all-or-nothing verifier re-checks everything built so far.

The hardest commonly-failed requirements are the reporting and provenance layers: sharding-report and time-range-observed (10/10), then dedup-policy, strict-late-audit, first-seen-epoch and summary-schema-out (9/10). run-provenance tops most fail lists (19 cases) — outputs must stay byte-identical across reruns. The reasons split between hard exits and silent wrong values: sharding-report and time-range-observed often produce empty output; provenance-strict and sharding-histogram exit 2 ("rc=2") where a report was expected; rates-validation should exit 2 on a non-numeric FX rate. Most revealing are the query cases where models run cleanly but compute the wrong money — "total=19.09" instead of "21.00" — a currency-normalization/rounding bug that returns rc=0 with a plausible-but-wrong answer.

Performance gap

This is a one-model task. Opus-4.8 reaches 9/13 rounds at 99% case pass, losing only a few late cases (rates-validation, sharding-report, time-range-observed). The drop-off is immediate and severe: GPT-5.5 is second at 4/13 but only 39% case pass, and Qwen3.6-Plus and GLM-5.1 manage 1/13 at ~26%.

The remaining six models all score 0/13 rounds, and unlike some tasks the case rates confirm genuine weakness rather than hidden progress — Kimi-K2.6 28%, MiniMax-M3 38%, DeepSeek-V4-Pro 21%, Qwen3.7-Max 46%, MiniMax-M2.7 22%, and DeepSeek-V4-Flash just 7%. Several even fail the build requirement ("/app/eventq missing" — the binary was never produced). Everyone outside Opus collapses on the same sharding/provenance reporting core, so the gap is less about which subtle requirement a model misses and more that only Opus sustains a correct, reproducible pipeline across all 13 rounds; the field's 99% → 39% cliff after first place is the widest top-two separation in this group.
